Pregunta 1:
Un algoritmo es una lista de pasos ordenados para resolver un problema.
Los Pasos para programar: 
* Entender: Leer bien el problema y saber que datos entran y salen
* Planear: Escribir los pasos en papel (pseudocodigo)
* Codificar: Pasar la logica al lenguaje de programacion (C++)
* Probar: Correr el programa con diferentes datos para ver si falla


Pregunta 2:
Un puntero es una variable especial que guarda la direccion de memoria de otra variable
* El operador & (direccion) sirve para saber en que parte de la memoria esta guardada una variable
* El operador * (desreferencia) sirve para ver o cambiar el valor que esta dentro de esa direccon de memoria


Pregunta 3:
Un arreglo es una lista fija que guarda varios datos del mismo tipo bajo un solo nombre
* Declarar: Se pone el tipo, el nombre y el tamano en corchetes,  ejemplo: int numeros[5];
* Acceder: Se usa el nombre y la posicion (indice) empezando desde cero,  ejemplo: numeros[0] = 10;


Pregunta 4:
El alcance es el lugar del codigo donde una variable existe y se puede usar.
* Esta la Variable local: Se crea dentro de una funcion (o llaves {}) y solo funciona ahi dentro. 
Y la Variable global: Se crea afuera de todas las funciones y cualquier parte del codigo la puede usar.


Pregunta 5:
* El private: Protege los datos para que nadie los cambie por error desde afuera de la clase.
* El public: Permite que cualquiera use las funciones de la clase.
* Encapsulamiento: Significa ocultar los datos internos de un objeto y solo mostrar lo necesario mediante funciones seguras.
